该项目是我老婆给我提的需求. 她在工作中常常需要正常会议纪要, 所以需要有一个能把录音转成文字的工具.

这里使用科大讯飞提供的 [讯飞开放平台](https://www.xfyun.cn/) 语音转写功能做了一些封装.
